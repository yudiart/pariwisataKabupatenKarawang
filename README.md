## Project & Requirement

**MERN STACK**
* MongoDB
* Express
* NodeJS
* ReactJS 

### Runing Project
**start**
```
npm run start
```
**client**
```
npm run client
```
**dev**
```
npm run dev
```
**server**
```
npm run server
```
### PORT
**Client**
```
http://localhost:3000
```
